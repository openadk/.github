## 🚀 Quick Start - AI Development Agents in 2 Minutes

```bash
# Clone next to your repositories
git clone https://github.com/openadk/openadk.git

# Start Claude Code
cd openadk && claude
```

Then in Claude's CLI interface, type:
```
start
```

This initializes the multi-agent workspace and discovers all your repositories.

## What is OpenADK? AI Agent Development Toolkit

OpenADK (Open Agent Development Kit) is an **open-source AI development framework** that enhances Claude Code with specialized AI agents for professional software development. Build better software faster with AI agents handling architecture decisions, security reviews, test creation, and DevOps automation.

### 🎯 Key Features - Enterprise AI Development for Everyone

- **🤖 9 Specialized AI Agents** - System architect, security expert, test engineer, DevOps engineer, and more
- **📁 Multi-Repository Management** - Work across microservices, monorepos, and multi-project workspaces
- **🔒 AI Security Reviews** - Automated vulnerability scanning and compliance checks
- **🧪 AI-Powered Testing** - Test generation, TDD workflows, and coverage analysis
- **🏗️ AI Architecture Design** - Technology decisions and system design with AI assistance
- **⚙️ DevOps Automation** - Infrastructure as code, CI/CD, and deployment with AI agents
- **📊 Code Quality Reviews** - Automated code review and best practices enforcement
- **🚦 Pre-commit Integration** - AI agents review code before commits
